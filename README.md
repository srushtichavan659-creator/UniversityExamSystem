🚀 University Exam System
A web-based University Examination Management System designed to streamline academic schedules, fee structures, exam centers, and performance tracking for IT students.

✨ Features Included
User Authentication & Text Input: Interactive forms for Login and Registration with clean input fields for student data.
Student Dashboard: Centralized panel showing student details (Exam Seat No: EXAM9074).
Live Status Sync: Integrated indicator representing MySQL Database Sync status.
Academic Schedules & Results: Quick access to exam dates and subject-wise score tracking (e.g., DBMS Mid Sem marks).
College & Centre Management: Clear views for checking allotted examination centers and campus guidelines.
Fee Structure: Integrated view for tracking examination fee details.

📂 Project Structure & Components
The repository contains the following core files:
index.html / login.html - The landing pages and text input forms for user authentication.
register.html - New user registration interface with form inputs.
dashboard.html - The central navigation hub after successful login.
schedule.html - Timetable and exam schedule details.
fees.html - Fee structures and breakdown.
results.html - Exam performance data and grading metrics.
college.html & centre.html - Administration, campus info, and designated test center locators.
style.css - Custom styling ensuring a clean, responsive web interface.

💻 Tech Stack
Frontend: HTML5, CSS3, JavaScript
Database Representation: MySQL Integration logic for real-time data representation
Hosting/Deployment: GitHub Pages
